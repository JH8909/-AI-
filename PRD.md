# AI 电商选品中台 — 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 产品定位
AI 电商选品中台是面向电商运营人员的内部工具，实现从产品数据采集、清洗、AI 评分、竞品分析，到内容生成、人工审核、数据复盘的完整链路。

### 1.2 MVP 范围（v0.1.0）

**包含：**
- 产品数据采集（CSV 导入 + 单链接解析）
- 数据清洗与标准化
- AI 产品评分（多维度打分）
- 竞品分析（同类产品对比）
- 内容生成（小红书文案、闲鱼文案）
- 人工审核队列
- 7 天数据复盘报告

**明确不包含：**
- 自动发布到小红书或闲鱼
- 批量自动化上架
- 自动私信/自动聊天
- 自动下单/自动采购
- 仿牌、医疗、减肥、保健品、三无产品相关
- 抓取需要登录或违反平台规则的数据

### 1.3 目标用户
- 电商运营人员（1-5 人团队）
- 需要选品决策支持的个体店主

## 2. 技术架构

| 模块 | 技术 | 说明 |
|------|------|------|
| 后台面板 | Next.js 14 + TypeScript | App Router, Shadcn/ui |
| 数据库 | Supabase (PostgreSQL) | Row Level Security |
| 自动采集 | Python + Playwright | CSV导入 + 单链接解析 |
| 自动化流程 | n8n | 工作流编排 |
| AI 分析 | LLM Provider Adapter | DeepSeek/GPT/Claude |
| 图片生成 | Prompt -> ComfyUI/即梦 | MVP 仅输出 Prompt |
| 内容排期 | 飞书多维表格 | 排期管理 |
| 人工审核 | 飞书审批 / 后台审核队列 | 双通道审核 |

---

## 3. 核心功能

### 3.1 产品数据采集
- CSV 导入：解析标准 CSV 格式产品列表
- 单链接解析：输入商品链接，Playwright 抓取
- 数据清洗：去重、格式标准化、敏感词过滤

### 3.2 AI 产品评分
评分维度（1-10 分）：
- 市场需求度
- 利润空间
- 竞争激烈度（越低越好）
- 合规风险（越低越好）
- 内容适配度
- 综合推荐指数

### 3.3 竞品分析
- 同品类产品价格对比
- 卖点差异化分析
- 内容策略对比
- AI 生成竞品分析报告

### 3.4 内容生成
- 小红书文案：种草文风，含标题、正文、话题标签
- 闲鱼上架文案：简洁卖点型
- 图片 Prompt：为后续图片生成准备

### 3.5 人工审核
- 审核队列：待审核内容列表
- 审核操作：通过 / 驳回 / 修改
- 审核记录：操作日志

### 3.6 7天复盘
- 选定产品池中产品，查看7天内数据变化
- AI 生成复盘报告

---

## 4. 安全与合规

### 4.1 内容红线
- 仿牌/山寨产品
- 医疗类产品
- 减肥类产品
- 保健品
- 三无产品

### 4.2 操作红线
- 自动发布到任何平台
- 自动下单/支付
- 抓取需登录的数据
- 违反平台 robots.txt

### 4.3 AI 输出规范
- 所有 AI 输出必须为 JSON
- 所有危险操作必须进入人工审核
- 敏感内容自动标记并拦截

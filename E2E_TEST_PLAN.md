## 闭环一：选品闭环 (E2E-1 ~ E2E-6)

### E2E-1 手动新增产品 -> 产品池展示

**测试路径:**
```
产品池 -> 点击"添加产品" -> 填写表单 -> 提交 -> 确认出现在列表
```

**输入数据:**
| 字段 | 值 |
|------|-----|
| 产品名称 | 测试铝合金手机支架 |
| 描述 | 可折叠便携铝合金材质 |
| 分类 | 家居 |
| 价格 | 29.90 |
| 成本 | 12.00 |
| 标签 | 手机支架,铝合金,桌面 |

**预期结果:**
1. [ ] 点击后弹出Modal表单
2. [ ] 所有必填字段可见(名称标*)
3. [ ] 分类下拉有10个选项
4. [ ] 提交后表单关闭
5. [ ] 新产品出现在产品池列表第一行
6. [ ] 数据正确: 名称/¥29.90/¥12.00/家居/csv_import/安全/草稿
7. [ ] 空价格产品显示"-"

**失败场景验证:**
1. [ ] 名称为空时提交按钮禁用或提示
2. [ ] 名称含"仿牌"自动标记为警告

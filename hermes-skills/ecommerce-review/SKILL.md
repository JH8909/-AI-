# 电商内容审核 Skill

## 适用场景
- AI 生成的内容需要人工审核
- 高风险产品需要二次确认
- 驳回内容需要修改后重新审核

## 审核原则

### 必须审核
- ✅ 所有 AI 生成的内容
- ✅ 所有 price_suggestion > 成本3倍的产品
- ✅ 所有 risk_level="warning" 的产品内容
- ✅ 所有首次上架的产品文案

### 审核标准

| 检查项 | 通过标准 |
|--------|----------|
| 产品合规 | 非仿牌/医疗/减肥/保健品 |
| 内容真实 | 无虚假宣传、无夸大功效 |
| 价格合理 | 建议价在市场范围内 |
| 文案质量 | 语句通顺、无错别字 |
| 标签准确 | 话题标签与产品相关 |

## 执行步骤

### 1. 查看审核队列
在后台面板 "审核队列" 页面查看待审核内容。

### 2. 审核操作
- **快速通过**：直接点击 "快速通过"（低风险产品）
- **审核通过**：填写意见后点击 "通过"
- **驳回**：填写驳回理由后点击 "驳回"（理由必填）

### 3. 处理后流程
- 通过 → 内容状态变为 `approved`，可用于排期
- 驳回 → 内容状态变为 `rejected`，可重新生成

## 风险提示
审核队列中带 ⚠ 标记的为高风险内容，需要重点关注：
- 产品分类为美妆但描述涉及功效
- 价格异常（过高或过低）
- 含敏感关键词但未完全匹配拦截规则

## 不可通过的红线
以下内容直接驳回，不可通过：
1. 含仿牌/山寨关键词
2. 宣称医疗效果
3. 涉及减肥/瘦身功效
4. 保健品相关
5. 任何形式的虚假宣传

# AI 电商选品中台 — 官方操作手册

> 版本: v0.3.0 | 2026-06-25
> 访问地址: http://localhost:3001

---

## 第一章：快速开始

### 1.1 启动系统

```bash
cd E:\hermes\电商自动化Agent\apps\admin-panel
node ..\..\node_modules\next\dist\bin\next dev -p 3001
```

浏览器打开 http://localhost:3001

### 1.2 配置 AI 能力（首次必做）

设置 → LLM 模型配置：
1. 填入 DeepSeek API Key
2. 选择模型（deepseek-chat 或 deepseek-reasoner）
3. 点击"保存配置"
4. 点击"测试连接"验证

> Key 获取: https://platform.deepseek.com/api_keys

### 1.3 配置数据库（可选）

设置 → 数据库配置：
- Supabase URL: https://xxx.supabase.co
- Supabase Anon Key: 从 Supabase 项目获取

> 迁移SQL: packages/supabase-schema/ 下执行 schema.sql + migrations/*.sql
> 未配置时自动使用本地缓存兜底，重启不丢失审核记录

## 第二章：标准操作流程

### 完整业务流

```ascii
1688链接一键提取 → AI 7维评分 → 内容资产包 → 审核清单通过 → 排期发布
      → 录入日数据(7天) → AI复盘 → AI决策 → 执行任务
```

### 2.1 产品导入

**入口：** 产品池

**方法一：一键提取1688链接（推荐）**
1. 点击"添加产品"
2. 在"1688链接一键提取"输入框粘贴商品链接
3. 点击"提取"按钮
4. 系统自动填充：产品名称、描述、分类、价格、标签
5. 确认后点击"保存"

**方法二：手动填写**
1. 点击"添加产品"
2. 填写名称(必填)、描述、分类(10选1)、价格、成本、标签
3. 点击"保存"

**方法三：CSV批量导入**
1. 点击"CSV导入"
2. 拖拽或选择CSV文件
3. 预览前10行数据
4. 查看导入统计（成功/跳过/拦截）
5. 点击"导入N条产品"确认

**CSV格式：**
```csv
产品名称,描述,分类,价格,成本,标签,链接
桌面理线收纳盒,硅胶桌面理线器,家居,25.00,8.00,桌面收纳,理线器,https://...
```

> 含仿牌/医疗/减肥/保健品关键词的产品自动拦截

### 2.2 AI 7维评分

**入口：** AI评分

| 维度 | 范围 | 说明 |
|------|------|------|
| 市场需求度 | 1-10 | 越高越好 |
| 利润空间 | 1-10 | 越高越好 |
| 竞争激烈度 | 1-10 | 越低越好 |
| 合规风险 | 1-10 | 越低越好 |
| 内容适配度 | 1-10 | 越高越好 |
| 供应链稳定性 | 1-10 | 越高越好（v0.3新增） |
| 视觉包装能力 | 1-10 | 越高越好（v0.3新增） |

**操作：**
1. 选择产品（下拉列表）
2. 查看5维进度条 + 2维新指标
3. 查看综合推荐指数和AI分析理由
4. 评分 >= 7 的产品显示"加入测试池"按钮
5. 点击后产品状态变为 testing_candidate，并自动创建占位内容草稿

### 2.3 结构化竞品分析

**入口：** 竞品分析

**输出内容（v0.3结构化格式）：**
- **推荐售价**：AI根据市场数据建议的定价
- **核心关键词**：5个推荐关键词
- **差评风险**：3条潜在差评风险点
- **内容角度**：推荐的内容营销切入角度
- 价格对比分析
- 差异化分析
- 内容策略建议
- 综合分析报告

**操作：**
1. 选择目标产品
2. 添加竞品链接（可选）
3. 点击"开始分析"

### 2.4 内容资产包生成

**入口：** 内容草稿

**一次性生成4项内容（v0.3新增）：**
- **小红书文案**：标题+正文+话题标签
- **闲鱼文案**：标题+正文+建议价格
- **图片Prompt**：中文版+英文版（可复制）
- **评论回复模板**：价格咨询/质量疑问/发货咨询

**操作：**
1. 点击"生成内容"
2. 选择产品 + 平台
3. 点击"生成"
4. 查看资产包4件套

### 2.5 审核队列（清单制）

**入口：** 审核队列

**v0.3 审核清单制：**
- 高风险内容（首次上架/含边缘词/价格异常）自动标记
- 高风险内容**禁止快速通过**
- 必须勾选全部5项检查项：
  - [ ] 产品合规：非仿牌/医疗/减肥/保健品
  - [ ] 内容真实：无虚假宣传
  - [ ] 价格合理：在市场范围内
  - [ ] 文案质量：语句通顺
  - [ ] 标签准确：话题标签相关

**操作：**
1. 浏览待审内容（高风险红色标记）
2. 点击"审核"展开审核面板
3. 高风险内容先勾选检查项
4. 输入审核意见（驳回时必填）
5. 点击"通过"或"驳回"

### 2.6 发布排期

**入口：** 内容草稿 → 已通过列表

1. 审核通过的内容显示"排期发布"按钮
2. 输入排期时间（格式：YYYY-MM-DD HH:mm）
3. 确认后内容状态变为"已排期"

### 2.7 发布记录与日数据

**入口：** 产品池 → 行操作按钮

**记录发布：**
1. 点击产品行的"发布"按钮
2. 选择日期、平台、发布链接
3. 保存

**录入日数据（数据库已扩展13个指标；当前表单开放4个基础指标）：**
1. 点击产品行的"日数据"按钮
2. 当前可填写：浏览量、点赞量、分享量、预估销量
3. 数据库已预留：收藏量、评论数、私信数、咨询数、成交数、成交额、利润、退款数
4. 保存（当天重复录入自动覆盖）

### 2.8 7天复盘

**入口：** 数据复盘

**自动决策（v0.3新增）：**
- 增长>50% 且 互动>10% → **magnify（放大投入）**
- 下降>30% 且 互动<3% → **eliminate（淘汰下架）**
- 其他 → **optimize（优化调整）**

**操作：**
1. 选择复盘产品
2. 查看4指标趋势卡（浏览量/点赞/分享/销量）
3. 查看7天明细表（当前显示浏览/点赞/分享/预估销量）
4. 点击"生成AI复盘报告" → 5段分析
5. 点击"生成决策建议" → 查看3种决策
6. 点击"应用决策" → 产品状态自动更新为 scale / optimize / rejected
7. 点击"生成任务" → 4条预设执行任务

## 第五章：常见问题

**Q：一键提取链接没反应？**
A：当前为 Mock 模式，自动匹配关键字返回模拟数据。对接真实1688需运行 Python Playwright 采集器。

**Q：数据录入重复怎么办？**
A：同一天同产品重复录入会自动覆盖（UPSERT）。

**Q：AI 返回结果不对？**
A：检查设置页的 DeepSeek API Key 是否配置正确，点击"测试连接"验证。

**Q：审核记录重启后丢失？**
A：v0.3 已修复。审核记录优先写入 Supabase，Supabase不可用时写入本地 .review-cache.json 文件，重启不丢失。
